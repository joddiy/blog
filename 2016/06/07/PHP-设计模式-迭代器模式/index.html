PHP 设计模式 - 迭代器模式

Jun 07, 2016 in Design pattern

前言

这是一个系列里面的一篇，主要介绍的一些 PHP 下的设计模式，大部分来自于 DesignPatternsPHP 项目，可能会加上《Design Patterns》里面的一些内容。因为都是英文版的，所以翻译的时候会出现一些偏差，不要期待太多噢！

目的

提供一种顺序获得聚合对象的每一个元素的方法，而且不用知道聚合对象的内部细节。

使用场景

  • 不用了解内部细节即可获得聚合对象的内容
  • 支持聚合对象的多重横切
  • 为聚合结构的不同横切方式提供了统一的接口

结构

参与者

  • Iterator
    • 定义了访问和横切数据的接口
  • ConcreteIterator
    • 实现了 Iterator 接口
    • 在聚合的切片中保存当前位置的记录
  • Aggregate
    • 定义了创建 Iterator 对象的接口
  • ConcreteAggregate
    • 实现了 Aggregate 接口
